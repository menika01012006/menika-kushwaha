# OOP Number Analyzer

A Java-based OOP project that performs different number analysis operations using Object-Oriented Programming concepts like encapsulation, inheritance, packages, and method reuse.

---

## 🚀 Features

This project can check:

- ✅ Palindrome Number
- ✅ Spy Number
- ✅ Strong Number
- ✅ Armstrong Number
- ✅ Disarium Number
- ✅ Reverse of Number
- ✅ Sum of Digits
- ✅ Count Digits
- ✅ Multiply Digits

## ⚡ How to Compile + Run (PowerShell)

From this folder (`OOPNumberAnalyzer`):

```bash
javac -d out (Get-ChildItem -Recurse -Filter *.java src | % FullName)
java -cp out Main
```
